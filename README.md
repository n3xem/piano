# piano
![ピアノ](https://raw.githubusercontent.com/wiki/n3xem/piano/images/piano-animation.gif)

# Description
3Dモデルのピアノです。  
Three.jsでピアノをモデリングし、Tone.jsで音を出しています。  
鍵盤をクリックするとその鍵盤に応じた音程の音を出すことが出来ます。 

# Requirement
node v10.14.2
# Install
```
$ npm install
```
# Usage
```
$ npm run test
```
そして、http://localhost:8080 にアクセスしてください。
